soffset¶

An unsigned byte offset added to the base address to get memory address.

警告

Assembler currently supports 20-bit offsets only. Use uimm20 instead of uimm21.

Size: 1 dword.

Operands: s, flat_scratch, xnack, vcc, ttmp, m0, uimm21
