hwreg¶

Bits d’un registre matériel auquel on est en train d’accéder.

Les bits de cet opérande signifient :

Bits

Description

5:0

ID du registre.

10:6

First bit offset (0..31).

15:11

Size in bits (1..32).

This operand may be specified as a positive 16-bit integer_number or using the syntax described below.

Syntax

Description

hwreg({0..63})

All bits of a register indicated by its id.

hwreg(<nom>)

All bits of a register indicated by its name.

hwreg({0..63}, {0..31}, {1..32})

Register bits indicated by register id, first bit offset and size.

hwreg(<nom>, {0..31}, {1..32})

Register bits indicated by register name, first bit offset and size.

Register id, offset and size must be specified as positive integer numbers.

Defined register names include:

Nom

Description

HW_REG_MODE

Shader writeable mode bits.

HW_REG_STATUS

Statut en lecture seule du nuanceur (shader).

HW_REG_TRAPSTS

Statut du piège.

HW_REG_HW_ID

ID de la vague, simd, unité de calcul, etc.

HW_REG_GPR_ALLOC

Allocation du SPGR et VGPR par vague.

HW_REG_LDS_ALLOC

Allocation LDS par vague.

HW_REG_IB_STS

Compteurs du nombre d’instructions en cours.

Exemples :

s_getreg_b32 s2, 0x6
s_getreg_b32 s2, hwreg(15)
s_getreg_b32 s2, hwreg(51, 1, 31)
s_getreg_b32 s2, hwreg(HW_REG_LDS_ALLOC, 0, 1)
